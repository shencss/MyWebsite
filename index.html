  • CSS Notes

    —— 个人积累的CSS笔记

    .......

    CSS Notes

    —— by shencss

    1. Border-box把border和padding都纳入总的大小之中（没有margin）IE盒子模型
    2. position：absolute会把元素变成块级元素，换行
    3. 按钮的cursor要设置为pointer
    4. transform-style：preserve-3D会保留和呈现子元素的3D位置
    5. transform-origin是设置旋转的基点
    6. perspective是视距，决定用户与Z轴之间的距离，perspective-origin决定视线的左右和上下位置
    7. @font-face字体格式：IE浏览器——EOT；Mozilla浏览器——OTF，TTF；Safari浏览器——OTF，TTF，SVG；Chrome浏览器——TTF，SVG
    8. cubic-bezier(x1,y1,x2,y2)用于animation自定义自己的速度曲线（cubic-bezier(.81, -0.62, 0.23, 1.92)实现蓄力回弹的速度曲线）
    9. animation-fill-mode控制元素在动画执行前和执行后的样式
    10. translate（100%）相对元素自身的宽高在X轴和Y轴上面移动
    11. img{width:100% height:100%}可以将img整个放入div中
    12. box-shadow:0px 0px 1px rgba(0,0,0,1)可以使得阴影四周分布
    13. opacity会使内容也透明，rgba可以解决这个问题
